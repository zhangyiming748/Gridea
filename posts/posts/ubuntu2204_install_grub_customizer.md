>目前 Grub-Customizer 没有进入 Ubuntu 的主仓库,可以通过添加 PPA 安装

```shell
$ sudo add-apt-repository ppa:trebelnik-stefina/grub-customizer
$ sudo apt update
$ sudo apt install grub-customizer
```
